1️⃣ Auth Routes ☑️

Method	   Endpoint	        Description 
POST	  /api/auth/register	Register admin (can be disabled after first use) ☑️
POST      /api/auth/logout      For logging out user ☑️
POST	  /api/auth/login	Login admin (returns JWT) ☑️
GET	  /api/auth/me	        Get logged-in user (verify token) ☑️



2️⃣ Project Routes

Method	    Endpoint         	Description
GET	       /api/projects	        Get all projects
GET        /api/projects/:id	    Get single project
POST	   /api/projects	        Create new project
PUT	       /api/projects/:id	    Update a project
DELETE	   /api/projects/:id	    Delete a project
